# 都道府県別人口統計ダッシュボード (e-Stat API Viewer)

本プロジェクトは、日本の都道府県別人口データを可視化するダッシュボードアプリケーションです。

## 概要

政府統計の総合窓口（e-Stat）の API を活用し、2020 年国勢調査に基づいた最新の人口情報をグラフィカルに表示します。
直感的な操作で、都道府県ごとの人口比較や地方内での人口比率を分析することが可能です。

## 主な機能

- **e-Stat API 連携**: API を通じて最新の公的統計情報を非同期に取得します。
- **インタラクティブ・チャート**: Chart.js を使用した動的な横棒グラフです。
    - 「標準（北から南）」および「人口順」の双方向ソート機能を搭載します。
- **プレイスカード表示**: 都道府県詳細データを視認性の高いカードグリッド形式で一覧表示します。
- **地方別詳細モーダル**:
    - 特定の都道府県を選択すると、所属する地方（関東、近畿等）の人口内訳を円グラフで表示します。
    - Vue の `Teleport` と `Transition` を活用した、アクセシビリティの高いUIです。

## 使用技術

- **Frontend**: Vue 3
- **Language**: TypeScript
- **Styling**: SCSS
- **Data Visualization**: Chart.js
- **Build Tool**: Vite
- **API**: 政府統計の総合窓口（e-Stat）API
- **IDE**: Antigravity（デザイン提案、ソースコード補完）

## 使用データについて

本アプリケーションで使用しているデータは、「政府統計の総合窓口（e-Stat）」の API 機能を利用して取得したものです。

- **出典**: 2020年 国勢調査（人口、人口動態及び世帯数）

## 今後の展望（拡張性）

- **レスポンシブデザインの高度化**: モバイル端末において、より最適化された表示パターンを導入します。
- **ダークモード対応**: CSS 変数を活用したテーマ切り替えを実装します。
- **時系列分析機能**: 過去の国勢調査データと比較し、人口増減を可視化する機能を追加します。
